Approximating a boxcar-shaped function, resulting from overlapping lights, with a differentiable function.

When considering a light as a circle, overlapping circles create a function that is not differentiable. If we think of the y-value as the brightness of the light, and consider two lights with the same brightness overlapping, the resulting function will have a box-like shape due to the overlapping brightness. If it is possible to approximate the resulting function of thousands of overlapping lights, with unknown positions and brightness levels, by a single differentiable function, then it would allow us to create a Glare Spread Function (which describes how light spreads) without having to draw each individual light.
